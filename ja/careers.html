採用情報

歯科医院に患者さまを呼ぶ、その仕組みを作る。

私たちは歯科専門の集患エージェンシーです。売っているのは一つだけ——チェアに座る患者さま。その裏側にある仕組み（即時のリード対応、AI受付、無断キャンセル回収）には、口で語るだけでなく、実際に作れる人が必要です。

自動化エンジニア（オートメーション実装担当）

GoHighLeveln8nAI音声＆チャット

あなたが作るもの

  • 昼夜を問わず、問い合わせに数秒で応答する、スピード・トゥ・リードのフロー
  • 見極め・予約し、患者さまをそのままクリニックのカレンダーへ入れる、AI受付の自動化
  • 来院前に患者さまを温め、来られなかった方を取り戻す、無断キャンセル対策のシーケンス
  • その土台となる“配管”：広告 → CRM → カレンダー → フォローアップを端から端までつなぎ、深夜2時でも止まらない信頼性で

こんな方に、向いています

  • GoHighLevel を使い込んでいる——パイプライン、ワークフロー、カレンダー、サブアカウント、スナップショット
  • n8n で構築できる（Make・Zapier でも可）。Webhook、API、多少の JSON に臆さない
  • AI音声・チャット（VAPI、Retell など）を組んだ経験があり、OpenAI API を理解している
  • 壊れないものを作り、壊れたものはデバッグできる
  • 作ったものを専門用語ではなく、平易な言葉で説明できる

あると、嬉しい経験

  • Twilio／メッセージング設定、到達性、A2P 10DLC 登録
  • クリニックや、その他予約制の地域ビジネスでの実績
  • 成功を「ワークフローが動いたか」ではなく、「患者さまを予約できたか」で測る
